Welcome to enhance

(Yet another) experimental 3D engine framework with render and basic support for implementing 3D applications.

News

  • 2016-08-28 - Ongoing changes to rendering and orientation modules. A trackball functionality was implemented on top of existing math utilities as well as UI eventing and event filtering modules. demo
  • 2016-08-20 - Ongoing refactoring of kernel modules. Several additions of UI and Filter modules enable keyboard and mouse event processing.
  • 2016-07-17 - Rendering code from previous branch was setup to run in the new infrastructure.
    This serves as the baseline for experimentation and changes.
